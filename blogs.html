Dom Selectors

Dom selectors are basically Javascript inbuilt functions which are used to select html elements with a document. In Javascript you can select the html elements using selectors in 5 ways.

  • document.getElementsByTagName()
  • document.getElementsByClassName()
  • document.getElementById()
  • document.querySelector()
  • document.querySelectorAll()

Now we will see how to use each of these selectors.

document.getElementsByTagName()

This method returns the collection of all the html elements with matching tag name.

Example - document.getElementsByTagName('li') will return all the list items in a collection. The developer then can traverse through that list using loops and access the elements.

document.getElementsByClassName()

This method returns the collection of all the html elements with specific class name mentioned while calling the function.

Example - document.getElementsByClassName('container') will return all the items having class container in a collection. The developer then can traverse through that list using loops and access the elements.

document.getElementsById()

This method returns the first element with the specific Id mentioned during calling the selector function.

Note - Keeping Id's unique is always a good practice.

Example - document.getElementsById('output') will return the first item having id output.

document.querySelector()

This method returns the first element with the specific tag, id or class name mentioned during calling the selector function.

Example - document.querySelector('input') will return the first item of the input tag.

document.querySelector(".container") will return the first element of class container.

document.querySelector("#output") will return the first element of id output.

document.querySelectorAll()

This method returns all the element with the specific tag, id or class name mentioned during calling the selector function.

Example - document.querySelector('input') will return all the items of the input tag.

document.querySelector(".container") will return all the elements of class container.

document.querySelector("#output") will return all the elements of id output.

That's All Folks...Isn't this easy :)

FormData Api In Javascript

FormData api provides an easy way to store the data collected from the html form. Formdata stores the key-value pairs for the mentioned html fields.

FormData provides a interface to define and access these fields disguised as set of key/value pair. Formdata object can be initialised by calling the constructor with the new operator.

const fd = new FormData();

FormData Methods

The FormData api provides various methods that can be performed on the objects

Some of the methods for FormData api are:

  • append() - This method appends a new key-value pair to the object

    • Example - fd.append("Name", "Lakshay")

  • delete() - This method deletes a key-value pair from the object

    • Example - fd.delete("age")

  • get() - This method retuns the first value for a given key

    • Example - fd.get("Name")

  • getAll() - This method returns all the values in an array for a given key

    • Example - fd.getAll("education")

  • has() - This method returns boolean value indicating if a key exists or not

    • Example - fd.has("designation")
